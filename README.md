# Medical Diary Project

Hệ thống Nhật ký Y tế (Medical Diary) tích hợp quản lý hồ sơ và chia sẻ dữ liệu an toàn.

## AI Agent Instructions
Để các AI Assistant (Antigravity, Cursor, Windsurf, v.v.) hoạt động hiệu quả và tuân thủ đúng kiến trúc dự án:
1. **Quy tắc cốt lõi**: Mọi Agent **PHẢI** đọc file `backend/docs/SYSTEM_DESIGN_SSOT.md` trước khi viết code.
2. **Project Rules**: Các quy tắc cụ thể đã được định nghĩa trong file `.cursorrules` ở thư mục gốc.
